    Reducing Interaction Overhead in Domestic Wireless Networks

    About

    MultiNet is a novel method for securely associating devices with a domestic wireless network. MultiNet creates novel architectural arrangement of the home network infrastructure: the network is dynamically re-configured to accept each pre-configured device, rather than the current norm where each device is configured to be acceptable to the pre-configured network. This is achieved by using a third device as an intermediary in the joining process, to create a visual out of band channel over which new devices can be introduced. MultiNet has several advantages over current methods:

    • Per device access revocation
    • Improved security without decreased usability
    • A Network Controller which provides an interface to the network

    Multinet consists of three parts. A modified version of hostapd designed to enable the creation and destruction of WPA2 secured WiFi networks without the need to restart; A set of RESTful APIs written in Python to modify the hostapd configuration and a Network Controller to provide a user interface to the network (currently implemented as a native Android app).
